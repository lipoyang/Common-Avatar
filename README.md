# Common-Avatar (Alpha Release)

- This project is a fork of [M5Stack-Avatar](https://github.com/stack-chan/m5stack-avatar).
- M5Stack-Avatar is a library dedicated to the M5Stack series.
- Common-Avatar is for any MCUs and Displays supported by LovyanGFX.
- Common-Avatar is depend on FreeRTOS or NuttX(SPRESENSE).
- You must define your display device I/F.
- For more details, see the document of M5Stack-Avatar.
